## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-10-01 | Spoofing | app/blueprints/api/auth.py | mitigate | require_api_token validates SHA-256 hash against DB; 256-bit entropy tokens via secrets.token_hex(32) |
| T-10-02 | Information Disclosure | audit logging | mitigate | Never log raw tokens; log only token_prefix (8 chars) or token.id in audit entries |
| T-10-03 | Information Disclosure | error responses | mitigate | API error handlers return generic messages; no stack traces in production; error_id for correlation |
| T-10-04 | Denial of Service | API endpoints | mitigate | Flask-Limiter per-token rate limiting (D-08); 429 with Retry-After header |
| T-10-05 | Tampering | token storage | mitigate | Raw tokens never stored; only SHA-256 hash persisted; token shown once at creation |
| T-10-06 | Elevation of Privilege | token management | mitigate | Token CRUD gated by @require_role("admin"); API tokens grant read-only access only |
| T-10-SC | Tampering | pip install flask-smorest | mitigate | Package Legitimacy Audit passed [OK] for all 4 packages (flask-smorest, marshmallow, webargs, apispec) — all marshmallow-code org |
</threat_model>
