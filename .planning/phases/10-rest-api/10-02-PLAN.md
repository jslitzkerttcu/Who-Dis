         d. Call `merger.merge_azure_ad_results(ldap_result, genesys_result, graph_result)` to get merged result
         e. Handle multiple results: if any source returns `multiple=True`, collect all entries into a list; otherwise wrap single result in list

         g. Build response: `{"data": paginated_results, "meta": {"page": page, "page_size": page_size, "total": total}, "errors": None}`

         i. Return the envelope dict
